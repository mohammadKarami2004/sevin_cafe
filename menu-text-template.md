# قالب متنی ساده — برای وقتی که نمی‌خوای گوگل‌شیت رو باز کنی

این روش پشتیبانه، برای وقتایی که یه تغییر بزرگ و یک‌جا داری (مثلاً کل منو رو
بازنویسی می‌کنی) و ترجیح می‌دی به‌جای گوگل‌شیت، یه لیست متنی بدی به AI تا
خودش فایل `js/data.js` یا `menu-template.csv` رو برات به‌روز کنه.

## فرمت

هر دسته با یه خط `#` شروع میشه. جلوش اسم دسته و آیکون (یکی از:
coffee / cold / herbal / dessert / breakfast / default). هر آیتم زیرش،
یه خط با این فرمت:

```
نام آیتم | قیمت | توضیح کوتاه (اختیاری) | ناموجود؟ (خالی بذار یا بنویس "ناموجود")
```

## مثال

```
# قهوه‌ها | coffee
اسپرسو | 65000
آمریکانو | 75000
لاته | 95000 | شیر بخارداده، لایه‌ی نرم فوم
کاپوچینو | 90000

# نوشیدنی‌های سرد | cold
آیس لاته | 100000
آیس آمریکانو | 80000

# دسر و شیرینی | dessert
چیزکیک نیویورکی | 120000
تیرامیسو | 115000 | | ناموجود
```

## چطور استفاده کنی

1. همین قالب رو خالی کن و منوی خودت رو توش بریز (یا با گوشیت بنویس، یا از
   روی منوی کاغذی رونویسی کن)
2. متن رو بده به یه AI (مثلاً همینجا) و بگو: «این لیست رو تبدیل به فرمت
   menu-template.csv کن» یا «این لیست رو با فرمت فعلی گوگل‌شیت من هماهنگ کن»
3. فایل CSV نهایی رو بگیر و توی گوگل‌شیت import کن (File → Import →
   Replace current sheet)، یا مستقیم بفرستش برای توسعه‌دهنده‌ت

اینجوری هیچ‌وقت لازم نیست خودت دستی وارد فرمت دقیق CSV یا کد بشی — فقط یه
لیست ساده‌ی خوانا می‌نویسی و AI ترجمه‌ش می‌کنه.
