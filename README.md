# منوی دیجیتال کافه سوین

## ساختار پروژه

```
sevin-cafe/
├── index.html          فقط ساختار HTML — استایل و منطق اینجا نیست
├── assets/
│   └── logo.jpg
├── css/
│   ├── tokens.css      همه‌ی رنگ‌ها و فونت‌ها — اول اینجا رو نگاه کن
│   ├── base.css        ریست و تنظیمات پایه
│   ├── hero.css        بخش بالای صفحه (لوگو، اسم، تگ‌لاین)
│   ├── nav.css         منوی چسبان دسته‌بندی‌ها
│   ├── menu.css        آیتم‌های منو و رنگ‌های هر دسته
│   ├── cart.css        نوار و کشوی سبد سفارش
│   └── footer.css
└── js/
    ├── data.js          ← تنها فایلی که برای کار روزمره لازمه ویرایش کنی
    ├── utils.js         فرمت قیمت و اعداد فارسی
    ├── cart.js          مدیریت وضعیت سبد (state)
    ├── menuRenderer.js  ساخت منو از روی data.js
    ├── cartUI.js        نمایش نوار/کشوی سبد
    ├── scrollSpy.js      هایلایت‌شدن منو هنگام اسکرول
    └── main.js          نقطه‌ی شروع، همه چیز رو به هم وصل می‌کنه
```

## برای کارهای روزمره

فقط **`js/data.js`** رو باز کن:

- **آیتم جدید** → یک خط به آرایه‌ی `items` همون دسته اضافه کن
- **دسته‌ی جدید** → یک آبجکت جدید به آرایه‌ی `MENU` اضافه کن (منوی بالا خودکار می‌سازتش)
- **تغییر قیمت** → فقط عدد `price` رو عوض کن
- **ناموجود کردن** → `soldOut: true` کن

نیازی به دست‌زدن به HTML یا فایل‌های دیگه نیست.

## برای تغییر رنگ‌بندی

فقط **`css/tokens.css`** رو باز کن — همه‌ی رنگ‌های سایت (طلایی، پس‌زمینه، متن) اونجا به‌صورت متغیر تعریف شدن. رنگ هر دسته‌ی منو (قهوه، سرد، دمنوش، ...) هم توی همون فایل، زیر بخش «per-category accent colors»ه.

## نکته‌ی مهم فنی: قبل از آپلود، مستقیم باز نکن

این پروژه از **ES Modules** (`import`/`export`) استفاده می‌کنه. مرورگرها به‌خاطر قوانین امنیتی، وقتی فایل رو مستقیم با دابل‌کلیک باز کنی (یعنی آدرسش با `file://` شروع بشه) ماژول‌ها رو بارگذاری نمی‌کنن و صفحه خالی می‌مونه.

**این یعنی چی برات؟**
- بعد از آپلود روی Netlify (یا هر هاست واقعی دیگه)، همه‌چیز کاملاً درست کار می‌کنه — نگران نباش.
- ولی اگه خواستی قبل از آپلود، روی سیستم خودت تستش کنی، نباید فایل `index.html` رو مستقیم باز کنی. یا مستقیم آپلودش کن و اونجا تستش کن، یا اگه دوست داشتی محلی تست کنی، به یکی بگو کمکت کنه یه سرور محلی ساده بالا بیاره (مثلاً با پایتون یا VS Code Live Server).

## دیپلوی

کل پوشه‌ی `sevin-cafe` (نه فقط `index.html`) رو روی Netlify درگ-اند-دراپ کن، یا به گیت‌هاب پیجز آپلود کن. لینک ثابتی می‌گیری که با QR Code وصلش می‌کنی به گوشیِ مشتری.
